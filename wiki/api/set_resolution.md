# set_resolution

**Syntax:** `set_resolution(width, height)`

**Description:**
Sets the output resolution of the current project in pixels. This configuration determines the dimensions of the final video file generated by the `build` command. If not specified, the project defaults to **1080p (1920x1080)**.

**Arguments:**
- `width` (Number): Desired output width in pixels (e.g., 1920).
- `height` (Number): Desired output height in pixels (e.g., 1080).

**Example:**
```scalar
// Configure the project for 4K output
set_resolution(3840, 2160)

// Enable 2D mode
Camera.set_mode_2d(true)
```

> [!IMPORTANT]
> To ensure the CLI correctly picks up the project resolution, `set_resolution` should be called at the very beginning of your Scalar script.
